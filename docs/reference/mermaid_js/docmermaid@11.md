# Mermaid – Free Open‑Source CDN

Mermaid is a JavaScript library that renders diagrams and flowcharts from text in a Markdown‑like syntax.  
This repository hosts the **jsDelivr CDN** for all released Mermaid versions.

> **NOTE** – The list below contains *every* released version of Mermaid that is available on jsDelivr.  
> Use the version that best matches your project’s requirements.

---

## 1. Quick Start

```html
<!-- Latest stable release -->
<script src="https://cdn.jsdelivr.net/npm/mermaid@11.9.0/dist/mermaid.min.js"></script>

<script>
  mermaid.initialize({ startOnLoad: true });
</script>

<div class="mermaid">
  graph TD;
    A-->B;
    B-->C;
</div>
```

> **Tip** – Replace `11.9.0` with any other version number from the list below to pin a specific release.

---

## 2. CDN URLs

| CDN Provider | URL Pattern |
|-------------|------------|
| jsDelivr | `https://cdn.jsdelivr.net/npm/mermaid@<VERSION>/dist/mermaid.min.js` |
| unpkg | `https://unpkg.com/mermaid@<VERSION>/dist/mermaid.min.js` |
| CDNJS | `https://cdnjs.cloudflare.com/ajax/libs/mermaid/<VERSION>/mermaid.min.js` |

> Replace `<VERSION>` with the desired release number.
