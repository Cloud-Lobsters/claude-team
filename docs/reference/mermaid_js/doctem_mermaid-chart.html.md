# Mermaid Chart – Official Documentation

> **Mermaid Chart** is a commercial extension of the open‑source Mermaid diagramming library.  
> It provides a web‑based editor, AI‑powered diagram generation, real‑time collaboration, and a plugin ecosystem that integrates with popular tools such as Confluence, Jira, VS Code, JetBrains IDEs, Google Docs, and Microsoft Office.

## About

Mermaid Chart was born from the Mermaid open‑source project and was founded by **Knut Sveidqvist** together with **Open Core Ventures**.  
The lead developers from Mermaid joined the company, ensuring a tight connection between the open‑source project and the commercial product.

## Features

| Feature | Description |
|--------|------------|
| **Editor** | Web‑based editor for creating and editing Mermaid diagrams. |
| **Mermaid AI** | Embedded AI chat that generates diagrams from natural‑language descriptions. |
| **Whiteboard** | Virtual whiteboard for collaborative diagram creation. |
| **Plugins** | Extend functionality via a plugin system. |
| **Collaboration** | Real‑time multi‑user editing (Pro & Enterprise). |
| **Comments** | Add comments to diagrams for discussion. |
| **Presentations** | Slideshow mode for viewing diagrams. |

## Getting Started

1. **Sign up** at <https://mermaidchart.com> (free account gives 3 diagrams).  
2. **Create** a new diagram in the web editor.  
3. **Use AI**: type a natural‑language description in the chat panel to generate a diagram.  
4. **Collaborate**: invite teammates to edit in real time.  
5. **Export**: download as PNG, SVG, or embed via Markdown.

## FAQ

| Question | Answer |
|---------|-------|
| **What is Mermaid?** | Mermaid is an open‑source diagramming tool that uses plain‑text syntax to generate SVG diagrams. |
| **Can I use Mermaid Chart offline?** | The web editor requires an internet connection; however, diagrams can be exported and used offline. |
| **Is my data secure?** | Yes. All data is encrypted in transit and at rest. See the [Security page](https://mermaidchart.com/security). |
